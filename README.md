## README - Membuat Website dengan crud api postman

### Cara Menjalankan Proyek Secara Lokal
1. Clone repositori ini ke komputer Anda:
   ```bash
   git clone https://github.com/username/nama-repositori.git
   ```
2. Buka folder proyek:
   ```bash
   cd nama-repositori
   ```
3. npm i
4. lalu , npm run dev
